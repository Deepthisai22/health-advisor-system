# 🩺 AI Health Advisory System

An AI-powered healthcare application that predicts diabetes risk and provides personalized health guidance using Machine Learning and Generative AI.

---

# 🚀 Features

✅ Diabetes Risk Prediction  
✅ AI-Generated Health Advice  
✅ AI Healthcare Chatbot  
✅ PDF Health Report Generation  
✅ Lifestyle Recommendation System  
✅ Interactive Streamlit Dashboard  
✅ Flask REST API Backend  
✅ Groq LLM Integration  

---

# 🧠 Technologies Used

## Machine Learning
- Scikit-learn
- Random Forest
- XGBoost
- Logistic Regression
- Decision Tree
- SMOTE

## Backend
- Flask
- REST APIs
- Python

## Frontend
- Streamlit

## Generative AI
- Groq API
- Llama 3 Model

## Other Libraries
- Pandas
- NumPy
- Joblib
- FPDF

---

# 📂 Project Architecture

User → Streamlit Frontend → Flask Backend → ML Model + Groq LLM

---

# 📊 ML Workflow

1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Handling Imbalanced Data using SMOTE
5. Model Training
6. Model Comparison
7. Best Model Selection
8. Model Saving using Joblib
9. API Development
10. Frontend Integration

---

# 🤖 AI Features

## Health Prediction
Predicts diabetes risk using patient health information.

## AI Health Advice
Provides personalized:
- diet recommendations
- exercise suggestions
- lifestyle precautions

## AI Chatbot
Users can ask healthcare-related questions and receive AI-generated responses.

---

# 📄 PDF Report

The system generates downloadable PDF health reports containing:
- prediction result
- diabetes risk level
- AI health recommendations
